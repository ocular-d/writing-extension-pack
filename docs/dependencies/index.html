    Prerequisite

    To enjoy the best experience, please make sure that you have Visual Studio Code, Node.js and all other dependencies installed.

    Font

    Hack is the default font.

    Ubuntu

    sudo apt install fonts-hack

    macOS

    brew tap homebrew/cask-fonts # Only once
brew cask install font-hack
