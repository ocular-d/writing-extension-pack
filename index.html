    (Technical) Writing With VS-Code

    About

    We ❤️ writing!

    Writing good and understandable documentation is as hard as writing good code.

    The aim of this project is to make it effortless to setup Visual Studio Code for (technical) writing by providing carefully tested extensions and settings.

    You can choose to install the extension pack, use the script or copy only settings you want.
